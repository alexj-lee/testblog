Alex is a graduate student in the Abbasi lab at UCSF (previously Seeley lab and Nano Precision Medical startup). Has a personal blog/website built with SvelteKit and Tailwind CSS, hosted with static assets in `/static/`. Uses animated protein structure visualizations (PDB models) as hero images on the homepage. The hero image is displayed at max-w-sm (384px) in Tailwind, so source images wider than ~480-768px are overkill for display purposes.
